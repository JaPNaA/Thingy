Syntax for **$** <br>
`$("a", b)`

#### for a
prefixes: . # (none) < \[ <br>
name: of the element or attribute <br>
end: > ] (none)

#### for b
Possible entiries:
<ul>
  <li> 1 / "first" / "element" </li>
  <li> 2 / "array" / "all" </li>
  <li> "last" </li>
  <li> "debug" / "debugger" </li>
</ul>
